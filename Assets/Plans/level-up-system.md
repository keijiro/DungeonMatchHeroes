# プロジェクト概要
- ゲームタイトル: Match-3 Combat Hybrid
- ハイレベルコンセプト: 7x7グリッドの下段を操作して連鎖を発生させ、リアルタイムで敵と戦うマッチ3パズル。
- プレイヤー: シングルプレイヤー
- ターゲットプラットフォーム: スタンドアロン (PC/Mac)
- レンダーパイプライン: URP
- UIシステム: UI Toolkit (UITK)

# ゲームメカニクス
## レベルアップシステム
- **累積経験値**: 経験値はリセットされず、常に累積値として扱われます。
- **必要経験値のスケーリング**:
    - 敵1匹の経験値を **6 XP** とします。
    - レベル1から2への到達: 敵約6匹分の経験値（**36 XP**）。
    - レベル10から11への到達: 敵約20匹分の経験値（**120 XP**）。
    - 成長曲線: レベル1から10までは線形に増加し、レベル10以降は20匹分の量（120 XP）で固定されます。
- **ステータス成長**:
    - 最大HP: レベル1の100から、**レベル10で200** まで線形に増加します。
    - レベルアップ時: HUDに反映される最大HPが増加し、**現在のHPは全回復** します。

## 経験値源 (調整後の数値)
- **敵の撃破**: 1匹あたり **6 XP**。
- **宝石 (マッチ3)**: 1ブロックあたり **1 XP**。
- **鍵**: 宝箱の解錠（またはマッチ）時に **15 XP**。

# UI
- **レベルアップ通知**: 画面中央に「LEVEL UP!」というメッセージが非ブロッキングでポップアップします。
- **ビジュアルスタイル**: 既存の `.center-message` USSクラス（「MONSTERS APPROACH!」と同じ）を使用します。
- **オーディオ**: レベルアップ時に `SEType.Victory` を再生します。
- **HUD**: 情報密度を上げないため、レベルの数値は表示しません。最大HPのバーとテキストは自動的に更新されます。
- **デバッグ**: 現在のレベルと最大HPをコンソールに出力します。

# キーアセットとコンテキスト
- `Assets/Scripts/CombatManager.cs`: プレイヤーのステータスと経験値処理のメインロジック。
- `Assets/Scripts/EnemyUnit.cs`: 敵の死亡時の経験値報酬ロジック。
- `Assets/UI/Main.uss`: 中央メッセージとHUDのスタイル定義。

# 実装ステップ
1. **CombatManager.cs の基本定数と変数の修正**:
    - `public int Level = 1;` を追加。
    - `BaseExp = 1;`、`KeyBonusExp = 15;` に数値を修正。
    - `private int GetThresholdForLevel(int level)` を実装: 特定のレベルに到達するために必要な累積経験値を、指定されたスケーリング（36 XP 〜 120 XP）に基づいて計算。
    - `private int GetMaxHPForLevel(int level)` を実装: レベルに基づいた最大HPを計算（L1で100、L10で200）。

2. **経験値加算ロジックの実装 (CombatManager.cs)**:
    - `public void AddExperience(int amount)` を作成:
        - `Experience` に加算。
        - `Level + 1` の閾値に達しているかチェックし、`LevelUp()` を呼び出し。
    - `private void LevelUp()` を作成:
        - `Level` をインクリメント。
        - 新しい `MaxHP` を適用し、**`CurrentHP = MaxHP;` で全回復** させます。
        - コンソール出力、効果音再生、UIポップアップを実行。

3. **EnemyUnit.cs の修正 (経験値報酬)**:
    - `Die()` 内で `CombatManager.Instance.AddExperience(6)` を呼び出し。

4. **アクション実行処理の更新 (CombatManager.cs)**:
    - `ExecuteAction()` 内の `PlayerExp` と `PlayerKey` の処理を `AddExperience()` に置き換え。

5. **UIポップアップの実装 (CombatManager.cs)**:
    - `private void ShowLevelUpPopup()` を実装。非ブロッキングなコルーチンで「LEVEL UP!」を表示。
